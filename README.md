# zombie-castle-survival
這個檔案是我的遊戲主檔案。  它的作用是讓瀏覽器知道要怎麼打開我的遊戲、顯示遊戲畫面，以及執行遊戲裡的所有規則。玩家打開這個檔案後，就可以看到開始畫面，進入殭屍生存遊戲，建造炮台、召喚護衛，保護城堡不被殭屍打倒。  這個檔案裡面包含了遊戲畫面、角色、殭屍、炮台、金錢、關卡、攻擊、升級和 Boss 出現的設定，所以它就像是遊戲的大腦。如果沒有這個檔案，GitHub Pages 就不知道要顯示什麼，玩家也沒有辦法進入遊戲。  把它改名成 index.html 之後，GitHub Pages 就會把它當成網站首頁。別人點開我的遊戲網址時，第一個看到的就是這個遊戲。
